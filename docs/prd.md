# 书签有效性监测网站需求文档

## 1. 应用概述

### 1.1 应用名称
书签有效性监测网站

### 1.2 应用描述
一个用于监测浏览器书签有效性的网站工具,支持上传多种格式的书签文件(如 HTML、Markdown 等),自动检测书签中链接的有效性,并展示检测结果,支持导出无效网站列表。

## 2. 核心功能

### 2.1 书签上传
- 支持上传浏览器导出的 HTML 格式书签文件
- 支持上传 Markdown 格式书签文件
- 支持其他常见书签格式的上传

### 2.2 链接有效性检测
- 自动解析上传的书签文件,提取其中的网址
- 检测每个链接的有效性
- 标识出无效链接(无法访问的网站)

### 2.3 结果展示
- 显示所有检测的书签列表
- 标注每个书签链接的有效性状态
- 展示无效链接的详细信息

### 2.4 无效结果导出
- 支持导出检测出的无效网站列表
- 导出格式包含无效链接及其相关信息

## 3. Markdown 书签格式说明

支持检测以下 Markdown 书签格式:

```
# 分类名称

| 名称 | 链接 | 介绍 |
| ---- | ---- | ---- |
| 网站名称 | https://example.com | 网站介绍信息 |
| 另一个网站 | https://another.com | 该网站的描述 |
```

注:Markdown 表格需包含名称、链接两列,支持可选的介绍列。
